# om-implement-feature-multi

> 🧑‍💻 Interactive — acts once, may ask questions, hands control back

Thin alias that runs [om-implement-feature](om-implement-feature.md) with the `multi` profile: the specification and the final diff are each reviewed by a fresh Claude context plus every configured independent advisor on one immutable packet.

## Parameters

Same as [om-implement-feature](om-implement-feature.md); the profile is fixed to `multi`.

## Works with

Wraps [om-implement-feature](om-implement-feature.md). Requires reviewers configured through [om-setup-agent-harness](om-setup-agent-harness.md).
