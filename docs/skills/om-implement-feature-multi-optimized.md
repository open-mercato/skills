# om-implement-feature-multi-optimized

> 🧑‍💻 Interactive — acts once, may ask questions, hands control back

Thin alias that runs [om-implement-feature](om-implement-feature.md) with the `multi-optimized` profile: configured workers build the bounded packets and the full advisor council reviews spec and diff. The worker's own model family never counts as independent review.

## Parameters

Same as [om-implement-feature](om-implement-feature.md); the profile is fixed to `multi-optimized`.

## Works with

Wraps [om-implement-feature](om-implement-feature.md). Requires both a worker and reviewers from [om-setup-agent-harness](om-setup-agent-harness.md).
