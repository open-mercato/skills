# om-fix-issue-optimized

> 🧑‍💻 Interactive — acts once, may ask questions, hands control back

Thin alias that runs [om-fix-issue](om-fix-issue.md) with the `optimized` profile: bounded, file-disjoint implementation packets are dispatched to a configured worker model in an audited sandbox, while the host keeps planning, integration, and every quality gate.

## Parameters

Same as [om-fix-issue](om-fix-issue.md); the profile is fixed to `optimized`.

## Works with

Wraps [om-fix-issue](om-fix-issue.md). Requires an audited worker binding from [om-setup-agent-harness](om-setup-agent-harness.md).
