# om-fix-issue-multi-optimized

> 🧑‍💻 Interactive — acts once, may ask questions, hands control back

Thin alias that runs [om-fix-issue](om-fix-issue.md) with the `multi-optimized` profile: configured workers implement the bounded packets and the full advisor council reviews the result. A reviewer from the worker's own model family counts as a self-check, never as independent confirmation.

## Parameters

Same as [om-fix-issue](om-fix-issue.md); the profile is fixed to `multi-optimized`.

## Works with

Wraps [om-fix-issue](om-fix-issue.md). Requires both a worker and reviewers from [om-setup-agent-harness](om-setup-agent-harness.md).
